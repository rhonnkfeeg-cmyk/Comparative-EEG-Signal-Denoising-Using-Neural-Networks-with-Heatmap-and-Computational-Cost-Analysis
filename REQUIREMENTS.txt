SOFTWARE AND SYSTEM REQUIREMENTS

--------------------------------------------------
PYTHON ENVIRONMENT
--------------------------------------------------

- Python version: 3.7
- TensorFlow version: 2.10

--------------------------------------------------
GPU AND CUDA SUPPORT
--------------------------------------------------

- NVIDIA GPU: GTX 1650 (or equivalent)
- Minimum VRAM: 4 GB

- CUDA Toolkit version: 11.2
- cuDNN version: 8.1

GPU acceleration is recommended for training (Test B),
but CPU execution is supported.

--------------------------------------------------
MATLAB CONFIGURATION
--------------------------------------------------

To ensure correct execution, the project directory must be
added to the MATLAB path.

1. Open MATLAB.
2. Navigate to the project root directory.
3. Add the project folder and all subfolders to the MATLAB path
   using one of the following methods:

   Option A (Command Window):
      addpath(genpath(pwd))
      savepath

   Option B (GUI):
      - Home â†’ Set Path
      - Add with Subfolders
      - Select the project root directory
      - Save

4. Verify that all required MATLAB scripts are accessible
   before execution.
